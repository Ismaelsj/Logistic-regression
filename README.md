	Riquieres:

		- pandas
		- matplotlib

Usage :

	- python3 logreg_train.py [-v][-f]

	- python3 describe.py
	- python3 histogram.py
	- python3 scatter_plot.py

Use '-v' to viusalize cost, '-f' to viusalize features importance.
